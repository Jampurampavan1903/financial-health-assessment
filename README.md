# Financial Health Assessment

## Project Overview
This project analyzes a user's monthly cashflow data to assess overall financial health.
It calculates savings rate, expense ratio, health score, and trends, and presents the results
using an interactive dashboard.

## Tech Stack
- Frontend: Streamlit
- Backend: FastAPI
- Data Processing: Pandas
- Deployment: Streamlit Cloud (Frontend), Render (Backend)

## Features
- Upload monthly cashflow CSV
- View overall financial summary
- Monthly breakdown with health score
- Interactive income vs expense charts
- REST API for financial analysis

## Live Application
Streamlit App:
https://financial-health-assessment.streamlit.app/

Backend API:

Swagger UI available at:

## How It Works
1. User uploads a CSV file
2. Frontend sends data to FastAPI backend
3. Backend computes financial metrics
4. Results are visualized in Streamlit
